Este repositório reúne materiais produzidos ao longo da disciplina de Banco de Dados, incluindo exercícios, atividades práticas e projetos. O conteúdo percorre todas as etapas, desde a modelagem até a implementação de sistemas de armazenamento de informações.

Conteúdos Abordados:
- Modelagem de Dados
Modelo Conceitual: definição de entidades, atributos e relacionamentos, representados em diagramas Entidade-Relacionamento (DER).

Modelo Lógico: conversão do modelo conceitual em tabelas, com definição de chaves primárias e estrangeiras.

Modelo Físico: estrutura definitiva do banco, especificação de tipos de dados e criação das tabelas.

- Bancos de Dados Relacionais
Organização em tabelas.

Relacionamentos (1:1, 1:N, N:N).

Garantia da integridade referencial.

Principais operações em SQL: SELECT, INSERT, UPDATE, DELETE.

- Normalização
Aplicação das formas normais (1FN, 2FN, 3FN).

Identificação e remoção de redundâncias.

Aumento da consistência e integridade dos dados.

- Objetivos da Disciplina
Entender como os dados são estruturados e armazenados.

Criar modelos eficientes aplicáveis a sistemas reais.

Interpretar e implementar diferentes níveis de abstração.

Exercitar fundamentos que sustentam aplicações modernas.

- Tecnologias e Ferramentas
Linguagem SQL.

SGBDs como MySQL, PostgreSQL ou similares.

Ferramentas de modelagem: brModelo, Draw.io, entre outras.

- Conteúdo do Repositório
Diagramas conceituais (DER).

Modelos lógicos.

Scripts SQL.

Exercícios de normalização.

Estudos de caso.
